## 第 3 页

# 1 总则

1.1 为适应温州城市交通发展新情况和停车需求新变化，加强市区建筑工程配置停车场（库）的规划建设与管理，缓解停车供需矛盾，改善城市静态交通环境，结合温州社会经济状况、城市车辆发展实际情况和未来发展趋势，制定本标准。

1.2 本标准适用于温州市辖区范围和瓯江新城核心区范围，其中温州市辖区包括鹿城区、龙湾区、瓯海区、洞头区行政辖区。温州市域范围内其他地区可参照执行。

1.3 本标准停车配建指标为建筑工程规划设计和审批时需要达到的最低标准。按照建筑面积计算建筑物配建停车位的，建筑面积应包括地上和地下空间有实际使用功能且有停车需求的面积。

1.4 各类新建、改建和扩建建筑工程配置停车场（库）应符合本规定要求。历史文化保护区（历史文化街区）、文物保护单位及其建筑控制范围等特殊区域内的建设项目，其停车配建指标按照批准的相关规划及规范标准执行。

1.5 城市建筑工程配建停车泊位及相关规划建设要求除应执行本标准外，还应符合国家、浙江省和本市现行的有关法律、法规和强制性标准的规定。

## 第 11 页

# 3 停车规划管理

## （1）停车配建通则

3.1 既有建筑改变使用功能或临时改变使用功能，应按改变后的使用功能计算停车配建指标。对于改、扩建建筑工程应至少满足按改、扩建建筑部分计算的停车配建指标。

3.2 各建筑工程需设置地面停车的，应集中安排用地，并做好人车分流。重点风貌管控区域应考虑地面停车位对景观风貌的影响，应适度集中并结合景观统筹设计布局。

3.3 建筑工程机动车停车场（库）的出入口车道及数量应符合表14的规定。当受场地条件限制，地下车库出入口数量不能满足表14的规定时，出入口可采用坡道与机械式垂直升降梯组合的形式，一台机械式升降梯服务的车辆数不应大于25辆，升降梯出入口外停候区应设置不少于2个车位的等候空间。

表14 机动车停车场（库）出入口车道及数量

| 机动车停车位当量数 | 机动车出入口车道及数量 |
| :--- | :--- |
| ≤100个* | 应至少设1个双车道出入口或2个单车道出入口 |
