## 第 3 页

# 二、申报要求

（一）创建主体：试点项目所在县（市、区）人民政府或开发区（新区）管委会。

（二）申报数量：原则上每个创建主体只能申报1个，最多2个。

（三）申报材料：创建主体以文件形式提出申请，并附《浙江省未来社区建设试点项目申报表》。各创建主体应明确实施主体，由实施主体根据《浙江省未来社区建设试点申报方案参考大纲》编制浙江省未来社区建设试点申报方案。申报方案应达到城市设计深度，并符合下列条件：

1. 明确回迁安置居民数、计划引进人才数两项综合指标和《浙江省未来社区试点创建评价指标体系（试行）》有关内容。

2. 申报方案应满足《浙江省城市设计管理办法》（浙建〔2017〕7号）及《浙江省城市设计编制导则》中详细城市设计、实施性城市设计的深度要求。其中，规划单元应达到详细城市设计深度；实施单元应达到实施性城市设计深度，并且参考《建筑工程设计文件编制深度的规定（2016年版）》中方案设计阶段有关要求，对建筑总平面、单体平面进行意向设计。

3. 申报方案应对九大场景开展初步策划，明确各场景的目标愿景、功能业态、设施配置标准、机制创新举措、组织方式等内容，并形成九大场景集成解决方案，落实33项指标体系要求。

4. 申报方案相应规划技术指标应符合地块控制性详细规划要求或能够通过调整控制性详细规划得到落实。
